# Casio Loopy Schematics

Schematics for various parts of, and accessories for, the Casio Loopy.  
These are provided in KiCad (9.0) schematic file format without PCB data.

**There is an error in the original design for the Power/AV board; C504 polarity is wrong. Install replacements the other way round or omit it.**

- Power/AV Board which includes voltage regulation and video switching, also a list of components
- Game cartridge PCBs used by the main 10 games
- Standard gamepad & mouse controllers
- Complete audio path separate from everything else, intended for analyzing audio levels, frequency responses of filters etc.

There is also a dimensional drawing of the original controller connectors, both cable-side and console-side.
